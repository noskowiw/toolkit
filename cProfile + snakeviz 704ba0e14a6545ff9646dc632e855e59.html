cProfile + snakeviz

date:

author: Wojciech Noskowiak

cProfile is a built-in Python module used for profiling the performance of Python code. It provides information about the amount of time spent in each function and how many times a function is called during the execution of the program.

snakeviz is a web-based viewer for the output of Python's cProfile module. It provides an interactive visualization of the profiling data, which makes it easy to identify performance bottlenecks in the code.

Using cProfile with snakeviz is a powerful combination for analyzing the performance of Python code. To use them together, you first need to run your Python program with cProfile enabled, which will produce a profiling file. You can then load this file into snakeviz, which will generate an interactive visualization of the profiling data.

The visualization provides various views, including a call tree view, a flame graph view, and a table view, which allow you to drill down into the profiling data and identify performance bottlenecks. You can also filter the data by function name or by the amount of time spent in a function.

By using cProfile and snakeviz, you can quickly and easily identify performance bottlenecks in your Python code and optimize it for better performance.

lets assume that you have a python script script.py that you want to profile. you first need to run a command: 

python -m cProfile -o output.prof script.py

after that you can view the visualized output with:

python -m snakeviz .\output.prof --server

The command will generate a link to a locally hosted site that presents a breakdown of function calls made while calling the script. To learn ho w to interpret the result consult the documentation:

https://jiffyclub.github.io/snakeviz/#controls
