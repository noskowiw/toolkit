Storing Data properly

Due to the design differences between the technologies used in order to make them work, storing data on a hard disc drive is more energy efficient that storing it in flash storage

Keeping that data accessible quickly also requires the running drive to be constantly turned on,.

Only the most frequently accessed data should be stored in faster storage tiers.
