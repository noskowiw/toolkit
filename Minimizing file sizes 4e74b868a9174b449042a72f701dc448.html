Minimizing file sizes

date:

author: Wojciech Noskowiak

  • when designing a website make sure files used by the site (especially pictures and videos ) are as small as they can practically be
  • compressed files are always better (png, jpg) (for example .bmp or .img formats are uncompressesd)
  • you can always use online converters/ compressors
  • simple graphics are always better than pictures
  • svg’s can be even better
  • use svg compressors
