Requests-Cache is a Python library that allows you to cache the results of your HTTP requests. This can be useful if you are making multiple requests to the same endpoint, or if you are working with data that is not updated frequently. By caching your requests, you can reduce the amount of time it takes to fetch data from a remote server, and improve the performance of your application.

Itâ€™s implemented as a simple drop in replacement for requests.Session backend, and can be implemented seamlessly into an existing python script that uses the requests package to send requests.

consider this fragment of code 

import requests

session = requests.Session()
for i in range(60):
    session.get('https://httpbin.org/delay/1')

in this case implementing caching is as simple as changing the library

import requests_cache

session = requests_cache.CachedSession('demo_cache')
for i in range(60):
    session.get('https://httpbin.org/delay/1')

Here this simple change increases the performance almost a 100 times
